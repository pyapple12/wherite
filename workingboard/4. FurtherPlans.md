# 路线图

> 当前版本: ver 0.11

本文档记录 Wherite 项目的未来计划和开发路线图。

---

## [F01].[短期计划]：（1-3 个月）

### [F01a].[PDF 导出功能]

#### [F01a01].[实现基础 PDF 导出] → [P05a]

目标内容：实现文章导出为 PDF 格式的基础功能

规划：

- 研究并选择合适的 Go PDF 生成库
- 实现基本的 PDF 导出功能
- 支持导出当前编辑的文章为 PDF 文件
- 确保导出的 PDF 文件包含文章标题和内容

#### [F01a02].[优化 PDF 样式和布局]

目标内容：优化 PDF 导出的样式和布局

规划：

- 设计美观的 PDF 模板
- 支持自定义字体和样式
- 优化页面布局和边距
- 确保导出的 PDF 文件具有良好的可读性

#### [F01a03].[添加高级导出选项]

目标内容：添加高级导出选项

规划：

- 支持导出范围选择（全部/部分内容）
- 添加页眉/页脚配置选项
- 支持自定义页面大小和方向
- 实现导出进度显示和取消功能

### [F01b].[打包和分发优化]

#### [F01b01].[实现自动化构建] → [P05b]

目标内容：实现自动化构建和打包流程

规划：

- 配置 CI/CD 流程
- 实现自动化测试和构建
- 支持多平台编译
- 生成各个平台的安装包

#### [F01b02].[编写用户文档]

目标内容：编写详细的用户手册和安装指南

规划：

- 创建安装和使用文档
- 编写功能说明和教程
- 添加故障排除指南
- 提供在线帮助文档

#### [F01b03].[添加版本更新检查]

目标内容：添加版本更新检查功能

规划：

- 实现版本检查机制
- 提供更新通知功能
- 支持自动下载和安装
- 优化更新过程的用户体验

### [F01c].[性能优化]

#### [F01c01].[数据库查询优化]

目标内容：进一步优化数据库查询性能

规划：

- 分析慢查询并进行优化
- 添加必要的数据库索引
- 优化查询语句和数据结构
- 提升大量文章时的查询速度

#### [F01c02].[Markdown 解析优化]

目标内容：提升 Markdown 解析和预览速度

规划：

- 优化 Markdown 解析算法
- 实现增量解析和渲染
- 减少不必要的重绘
- 提升大文章的编辑体验

#### [F01c03].[内存使用优化]

目标内容：优化内存使用，减少资源消耗

规划：

- 分析内存使用情况
- 优化数据结构和算法
- 实现资源回收机制
- 提升程序运行效率

## [F02].[中期计划]：（3-6 个月）
