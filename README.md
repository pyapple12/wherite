# Wherite - 文章查询系统

**版本：ver 0.01 技术先行版本**

## 项目简介

Wherite是一个基于SQLite数据库的文章查询系统，支持根据文章ID快速检索和显示文章内容。

## 功能特性

- 连接SQLite数据库
- 根据文章ID查询文章
- 显示文章标题和内容
- 命令行交互界面
- 输入验证和错误处理

## 技术栈

- **编程语言**: Go 1.25.4
- **数据库**: SQLite3
- **数据库驱动**: modernc.org/sqlite v1.26.0（纯Go实现，无需CGO）

## 数据库结构

### 表名：articles

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| title | TEXT | 文章标题 |
| content | TEXT | 文章内容 |

## 安装与运行

### 前置要求

- Go 1.25.4 或更高版本
- SQLite3 数据库文件 `wherite.sqlite3`

### 安装依赖

```bash
go mod download
```

### 运行程序

```bash
go run main.go
```

### 编译程序

```bash
go build -o wherite
```

## 使用说明

1. 运行程序后，系统会显示提示信息
2. 输入要查询的文章ID
3. 系统会显示文章的标题和内容
4. 输入 `0` 退出程序

### 示例

```
文章查询系统
请输入文章ID (输入0退出):
> 1

=== 文章内容 ===
ID: 1
标题: 测试文章1
内容: 这是第一篇测试文章的内容。
================

请输入下一个文章ID (输入0退出):
> 0
退出系统。
```

## 技术说明

### 为什么选择 modernc.org/sqlite？

本项目使用 `modernc.org/sqlite` 而非 `github.com/mattn/go-sqlite3`，原因如下：

- **纯Go实现**：无需CGO，不依赖GCC
- **跨平台兼容性**：在Windows、Linux、macOS上都能正常运行
- **编译简单**：无需配置CGO环境
